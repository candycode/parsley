Detailed Description

Implementation of IParser interface.

Allows value semantics with polymorphic types through the pImpl idiom:

  • an implementation of IParser is stored internally
  • all calls to IParser methods are forwarded to the containted IParser implementation the contained IParser instance is owned my Parser objects and deleted when Parser deleted. Note that contained parsers are never shared: new copies are created during constructors and assignment.
