Detailed Description

template<class ValidatorT = AlphaNumValidator>
class parsley::SequenceParser< ValidatorT >

Parser for strings.

It can be configured with validators having the following compile time interface:

struct ...
{
bool Validate( const String& currentlyValidated, Char newChar ) const;
void Reset();
};
