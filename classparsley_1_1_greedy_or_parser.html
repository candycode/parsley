Detailed Description

Or Parser: applies a set of parsers to the given stream and validates the input only of at least one of the applied parser validates the input characters.

The parser selected as the validating parser is the one that validates the maximum number of characters. At the end of the parsing operation the get pointer is positioned:

  • at the same position where it was before calling the Parser method if no parser that validates the input is found
  • one character past the last character validated by the parser that validates the maximum number of characters.
