Detailed Description

template<class IT, unsigned long RESIZE_THRESHOLD = 0xffffff, typename IT::char_type EOL_ = '\n'>
class parsley::InChStream< IT, RESIZE_THRESHOLD, EOL_ >

Input character stream class featuring automatic tracking of current line and char.

All method calls are forwarded to the equivalent methods in the contained input stream instance. Only get unget peek tellg and seekg are implemented.

Template Parameters
ITinput stream e.g. std::istream
RESIZE_THRESHOLDmax number of end of line separators to keep in memory
EOL_end of line character
