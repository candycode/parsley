Parsley

Introduction

Parsley is a multiplatform parsing framework originally developed to extract values from text data files. It can be used as it is or easily extended to support Parsing Expression Grammar it was built to be used in conjuntion with the provided state machine manager to ease the parsing of different sections of text files. It doesn't have any dependency other than STL and is developed and tested on Linux, Windows and Mac OS.

Sample usage

Integer parser

// 1) create input stream to read from
std::istringstream iss( "-123" );
InStream is( iss );
// 2) create parser and set the name for the returned values to 'number'
IntParser il( "number" );
// 3) apply parser: returns true on success
assert( il.Parse( is ) );
// 4) extract parsed value by name; returned values
// are instances of type Any which can be cast to any other type
// provided the conversion is possible
assert( il.GetValues().find( "number" )->second == -123 );

Compilation

Dependencies

No dependencies, the only C++ library used is STL.

Building

Building is performed through CMake. Should work with any version; tested with versions 2.6 and 2.8.
