Detailed Description

Float parser.

Parses number in signed/unsigned floating point number in plain or exponential format. Supports both E and D as exponent identifier. The decimal separator is assumed to be the '.' character. Can be extended to be locale-aware by reading the decimal separator through a call to 

 std::use_facet< std::numpunct< Char > >( is.getloc() ).decimal_point ();
